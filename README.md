## Installation

Use the package manager [npm](https://docs.npmjs.com/) to install.

```bash
npm install
```
### Compile
```bash
truffle compile
```
### Migrate
```bash
truffle migrate
```
truffle migrate / truffle migrate --network ropsten / truffle migrate --network kovan ...
### For Test
```bash
truffle test
```
## Launch 
```bash
npm run dev
```
then you can see website on your browser.
